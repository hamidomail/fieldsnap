Voici FieldSnap — une application mobile complète ! 📱

Fonctionnalités incluses
📷 Capture

Activation de la caméra du téléphone (caméra arrière en priorité)
Déclenchement avec effet flash visuel
Horodatage + coordonnées GPS gravés directement sur la photo

📍 GPS

Localisation automatique au démarrage
Bouton 🛰️ pour rafraîchir la position
Précision affichée (±Xm)
Simulation si l'accès est refusé

📝 Observation

Zone de texte pour saisir les notes terrain
Métadonnées affichées en temps réel (lat, lng, date)

🖼️ Galerie

Liste de toutes les observations avec photo miniature
Suppression individuelle
Partage natif (ou copie presse-papier)
Vue détail en modal

🗺️ Carte

Liste des points GPS enregistrés
Visualisation géographique

💾 Stockage

Persistance locale via localStorage — les données survivent aux fermetures
